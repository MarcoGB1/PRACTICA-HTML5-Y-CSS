Atributos HTML

No. Atributo Definición
1 alt Texto alternativo en caso de que la imagen no se pueda mostrar.
2 align Especifica el alineamiento horizontal del elemento.
3
4 height Especifica el alto de una imagen
5 width especifica el ancho de una imagen
6 style Especificar el estilo de un elemento
7 title Es agregado a un elemento p.
8 id Especifica un ID único para un elemento
9 href Dirección de enlace
10 src El nombre del archivo de una imagen.
11 loop Indica si el medio debe volver a ser reproducido desde el principio al llegar al final.
12 max Indica el máximo valor aceptado.
13 min Indica el valor mínimo aceptado.
14 rows Define el número de filas en un área de texto.
15 size Define el ancho del elemento (en píxeles).
